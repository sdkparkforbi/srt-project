## 2. 문서 구성 순서 (국문 논문)

1. **첫쪽(표지)**: 제목 / 저자(각주로 소속·직위 표시) / **국문초록(500자 이내)** / 주제어(5개 이하, 필요 시 영문 병기) / JEL
2. **본문**: 제목·저자 재기재 없이 **「Ⅰ. 서론」부터** 시작
3. **마무리쪽**: 참고문헌, 부록(필요 시)
4. **마지막 쪽**: 영문 제목 / 저자 영문 성명(각주 영문) / **영문초록(800단어 이내)** / keyword / JEL

> 최근호(제13권 제1호, 2026.4) 표지 실제 배치: `제목` → `부제(- …)` → `저자*, 저자**` → `국문초록` → `□ 주제어:` → `□ JEL:` → `투고일/수정일/게재확정일` → 하단 각주에 저자 소속·이메일·역할(제1저자/교신저자).

## 3. 번호 체계 (장·절·목)

| 수준 | 표기 | 예 |
|---|---|---|
| 節 chapter(장) | Ⅰ, Ⅱ, Ⅲ … | **Ⅰ. 서론** |
| 項 section(절) | 1, 2, 3 … | **1. 합성증권화의 구조** |
| 目 subsection(목) | (1), (2), (3) … | **(1) 우선손실 트렌치** |

- 번호 뒤 한 칸 띄우고 제목. 모든 장·절·목 번호와 제목은 **굵게**, 간결하게.
- 수식: 번호 `(1),(2)…`를 **우측 정렬**, 반드시 **수식편집기** 사용(일반 문장 금지).

## 4. 표·그림 (국문)

- 번호: `[표 1]`, `[그림 1]` … 일련번호.
- **번호·제목은 굵게·중앙정렬**, 번호와 제목은 한 줄에(자간 조정 권장).
- **설명은 제목과 표/그림 사이**에 배치. 본문 안 봐도 표/그림만으로 내용 전달되게, **자료 출처 표시**.
- 표·그림은 본문 해당 위치에 삽입(논문 끝으로 몰지 말 것).

> 우리 논문 그림 예: `[그림 1] 합성증권화 트렌치 구조`, `[표 1] 은행별 기업 평균위험가중치(2025년말)`, `[표 2] 이전비중별 RWA 경감 시나리오`, `[그림 4] 시스템리스크 전이 경로`. 각 표 아래에 "자료: 전국은행연합회 은행경영통일공시 「리스크관리」(2025년말); 금융감독원 FISIS" 명기.

## 5. 인용·참고문헌 양식 (핵심만)

**본문 인용**
- 국문 1인: `홍길동(1999)` (성명과 괄호 붙임)
- 국문 2인: `변진호·조성순(2011)` (가운데점 `·`)
- 국문 3인 첫 언급: `빈기범·정무권·윤정선(2011)` / 이후: `빈기범 외(2011)`
- 영문 1인: `Shleifer (1994)` (성명과 괄호 **띄움**)
- 영문 2인: `Shleifer and Vishny (1995)`
- 영문 3인 첫 언급: `LaPorta, Shleifer and Vishny (2001)` / 이후: `LaPorta et al. (2001)` (*et al.* 이탤릭)
- 기관: 국문 `한국은행(2002)`, 해외 `IMF (2011)` / `BCBS (2024)`
- 괄호 안 인용: `(빈기범, 2010)`, `(홍길동, 1999; 변진호·조성순, 2005)`

**각주(footnote)**: 미주 금지. 문장 끝 마침표 뒤 **붙여서** 번호. 저자소속 각주와 별개 일련번호.

**＜참고문헌＞**(장 제목과 동급 굵기) — 본문/각주에 인용된 문헌만. 한국문헌(가나다) → 동양문헌 → 영어문헌(last name ABC) → 서양문헌 순, 일련번호 부여.
- 따옴표 `" "`·겹꺾쇠 `『 』` 금지. 영문·서양 단행본/정기간행물명은 **이탤릭**.
- 국문 정기간행물 예: `김준한·윤병학, 2008, 뉴케인지안 모형에서의 통화의 역할과 예측력 검정, 금융연구, 제22권 제1호, 53-79.` (발행기관 미표시)
- 영문 정기간행물 예: `Case, Karl E. and Robert J. Shiller, 1989, The Efficiency of the Market for Single-Family Homes, American Economic Review 79, 125-37.` (저널명과 volume 사이 콤마 없음, volume까지만)
- 동일 저자 연속: 둘째부터 저자명을 공백선으로 대체. 동일 저자·동일 연도: `2003a, 2003b`.
